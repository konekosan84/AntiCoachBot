# AntiCoach Platform Product Requirements

## Vision
AntiCoach Platform — это система онлайн-записи и управления сервисным бизнесом нового уровня, вдохновленная YCLIENTS, но ориентированная на гибкость, автоматизацию и масштабирование. Платформа должна объединять инструменты записи клиентов, управления персоналом, аналитики и интеграций с внешними сервисами в одном месте.

## Целевые пользователи
- Салон красоты, барбершопы, спа и велнес центры
- Клинике и медицинские центры с частной практикой
- Образовательные центры и спортивные студии
- Небольшие сервисные бизнесы, где требуется управление расписанием и оплатами

## Ключевые пользовательские задачи
1. **Онлайн-запись** клиентов по множеству фильтров: услуга, специалист, дата, место, длительность, доступность оборудования.
2. **Онлайн-оплата** и предоплата через интегрированные платежные шлюзы.
3. **Визуальная кастомизация** клиентского портала: цвета, логотипы, фон, структуру блоков.
4. **Мультимедийное сопровождение** услуг и специалистов (фото, видео, отзывы).
5. **Расширенная аналитика** и отчеты по дням, услугам, специалистам, выручке, эффективности маркетинга.
6. **Интеграции** с CRM, бухгалтерией, мессенджерами, сервисами рассылок.
7. **Автоматические напоминания** и рассылки в WhatsApp, Telegram, email, SMS.
8. **Разграничение прав доступа** для владельцев, администраторов, специалистов и внешних партнеров.

## Основные сценарии
- Клиент выбирает услугу → видит доступных специалистов → подбирает время → оплачивает → получает подтверждение и напоминания.
- Владелец настраивает визуал виджета, фильтры и логику записи для разных точек.
- Администратор следит за расписанием, подтверждает заявки, ведет журнал посещений.
- Аналитик/менеджер выгружает отчеты для контроля KPI и принятия решений.

## Функциональные блоки
1. **Модуль расписания**
   - Многоточечное расписание с календарями специалистов
   - Наборы правил записи (буфер времени, окна, запреты)
   - Сложные фильтры поиска доступных слотов

2. **Модуль каталога услуг и товаров**
   - Иерархия категорий, доп. услуги и наборы
   - Медиа-контент: фото, видео, FAQ
   - Динамические цены (прайс-листы по времени/специалистам)

3. **Модуль клиентов и CRM**
   - Карточка клиента, история посещений, бонусы
   - Интеграции через webhooks, API, готовые коннекторы

4. **Модуль платежей**
   - Подключаемые провайдеры (Stripe, CloudPayments, YooKassa и др.)
   - Настройка предоплаты, рассрочки, безнал/наличный расчет
   - Финансовые отчеты и экспорт для бухучета

5. **Модуль отчетности и аналитики**
   - Дашборды по дням/неделям/месяцам
   - Срезы по услугам, специалистам, филиалам
   - Экспорт в Excel/CSV, API для BI-систем

6. **Модуль уведомлений**
   - Каналы: WhatsApp, Telegram, email, SMS
   - Триггеры: запись, отмена, перенос, задолженность, маркетинговые акции
   - Шаблоны с персонализацией и A/B тестами

7. **Модуль кастомизации**
   - Тема, шрифты, цвета, CSS, компоненты
   - Редактор drag&drop для клиентского портала
   - Разграничение стилей по филиалам и сайтам

## Нефункциональные требования
- Масштабируемая микросервисная архитектура с возможностью горизонтального масштабирования.
- Высокая доступность (99.9% SLA) и надежность хранения данных.
- Соответствие требованиям GDPR/ФЗ-152 (персональные данные), PCI DSS для платежей.
- Локализация интерфейсов (RU, EN, возможно другие языки).
- Открытый REST и GraphQL API, поддержка webhooks.
- Низкая латентность ответа (<200ms для большинства запросов).

## Минимальный жизнеспособный продукт (MVP)
- Управление услугами, специалистами и календарем с веб-интерфейсом администратора.
- Публичный виджет записи с выбором услуги и специалиста, подтверждением через email/SMS.
- Интеграция с одним платежным провайдером.
- Базовый отчет по посещениям и выручке.
- Интеграция с Telegram для уведомлений.

## Будущее развитие
- Поддержка маркетплейсов и агрегаторов услуг.
- AI-подсказки по оптимизации расписания и динамическому ценообразованию.
- Мобильные приложения для клиентов и специалистов.
- Голосовые ассистенты и интеграция с call-центрами.
